## hexo-theme-line

Dead simple and beautify Hexo theme.

## preview

![img](ad.png)

## how to use

1 clone this repo.
```
git clone https://github.com/zhaobao/hexo-theme-line.git themes/line
```

2 change theme to line.
```
theme: line
```

3 custom.
```
menus: change menu
zoom: image zoom support
disqus: disqus support
```
